Client-Server, o que é?

Quando entramos no vasto mundo de TI, é necessário entender como funciona a comunicação entre um dispositivo e um servidor;

Um servidor é um recurso dentro de um sistema computacional, capaz de processar aplicações, armazenar dados e prestar serviços, enquanto o cliente é um computador que solicita esses serviços e recursos. Em uma infraestrutura de TI, ambos formam a arquitetura conhecida como “cliente-servidor”.

A comunicação cliente-servidor é baseada em troca de mensagens, onde o cliente envia requisições e o servidor responde com as informações requisitadas.

  • No Head contêm dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados. Na prática, todo o conteúdo do cabeçalho fica delimitado entre a abertura e fechamento tag head.

Tags HTML, o que são?

As tags nada mais são do que identificadores em uma página HTML, servem para identificar um elemento e designar a ele uma função. Como exemplo temos as tags:

  • h1, h2, h3...h6 (servem para delimitar um título ou subtítulos)
  • p (serve para delimitar um parágrafo
  • i, strong, u (servem para formatação de texto)
  • ul, ol (servem para definir listas ordenadas e não ordenadas
